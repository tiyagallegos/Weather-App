# AJAX WEATHER LAB
## Technologies: 

```bash
Html, JS, CSS, JQuery
```
## API: [weatherAPI](http://api.openweathermap.org)

## Features: 

This app will provide following paramters for the city you type in:
Country, Temperature, Feels like, looks like, humidity, wind speed, sunrise time, and sunset time
